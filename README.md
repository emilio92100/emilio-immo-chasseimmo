# Emilio Immobilier — Outil de Chasse Immobilière

Outil web sur mesure pour Alexandre ROGELET, chasseur immobilier sur Paris & 92.

## Stack
- **Frontend** : Next.js 14 + TypeScript
- **Style** : CSS Modules + Variables CSS
- **Base de données** : Supabase
- **Email/SMS** : Mailjet
- **Hébergement** : Vercel

## Démarrage local
```bash
npm install
npm run dev
```
